# 우리동네신천지
쉽게 찾기 어려운 그들의 위치 정보를 알려주는 웹사이트입니다.  
사용자 위치정보와 주소 검색을 통해 그들의 거점 정보를 알 수 있습니다.

## 개발자
:satisfied: __최상현__(홍익대학교)  
:satisfied: __김재욱__(숭실대학교)  

## 사용 기술
* __JAVA Springboot__ - REST API 서버
* __Mariadb__ - 데이터베이스
* __JavaScript__ - UI 개발 및 네이버 지도 API 이용한 서비스 구현
* __HTML__ - UI 개발
* __CSS__ - UI 개발
